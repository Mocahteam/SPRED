========================================================================
    DLL DE CONTRÔLE ACTIVEX : vue d'ensemble du projet GeiOffice
========================================================================

ControlWizard a créé ce projet pour votre DLL de contrôle ActiveX GeiOffice qui
 contient 1 contrôle.

Ce squelette de projet décrit non seulement les bases de l'écriture d'un
contrôle ActiveX, mais est également un point de départ pour l'écriture de fonctionnalités
spécifiques à votre contrôle.

Ce fichier contient un résumé du contenu de chacun des fichiers qui
constituent votre DLL de contrôle ActiveX GeiOffice.

GeiOffice.vcproj
    Il s'agit du fichier projet principal pour les projets VC++ générés à l'aide d'un 
    Assistant Application. Il contient les informations sur la version de 
    Visual C++ qui a généré le fichier et des informations sur les plates-formes,
    configurations et fonctionnalités du projet sélectionnées avec l'Assistant Application.

GeiOffice.h
    Il s'agit du fichier Include principal pour la DLL du contrôle ActiveX. Il
    comprend d'autres fichiers Include spécifiques au projet, tels que resource.h.

GeiOffice.cpp
    Il s'agit du fichier source principal qui contient le code pour l'initialisation de la DLL,
    sa finalisation et autres opérations.

GeiOffice.rc
    Il s'agit d'une liste des ressources Microsoft Windows que le projet
    utilise. Ce fichier peut être directement modifié avec l'éditeur de ressources Visual
    C++.

GeiOffice.def
    Ce fichier contient des informations relatives à la DLL du contrôle ActiveX qui
    doit être exécutée avec Microsoft Windows.

GeiOffice.idl
    Ce fichier contient le code source ODL (Object Description Language) pour la
    bibliothèque de types de votre contrôle.

GeiOffice.ico
    Ce fichier contient une icône qui s'affichera dans la boîte de dialogue À propos de. Cette icône
    est incluse dans le fichier de ressources principal GeiOffice.rc.

GeiOfficeCtrl.cpp

GeiOfficePropPage.h

GeiOfficePropPage.cpp
    Ce fichier contient l'implémentation de la classe C++ CGeiOfficePropPage.

CGeiOfficeCtrl.bmp
    Ce fichier contient une bitmap qu'un conteneur utilisera pour représenter le contrôle
    CGeiOfficeCtrl lorsqu'il s'affiche dans une boîte à outils. Cette bitmap
    est incluse par le fichier de ressources principal GeiOffice.rc.

/////////////////////////////////////////////////////////////////////////////
Autres fichiers standard :

StdAfx.h, StdAfx.cpp
    Ces fichiers sont utilisés pour générer un fichier d'en-tête précompilé (PCH) 
    nommé GeiOffice.pch et un fichier de type précompilé nommé StdAfx.obj.

resource.h
    Il s'agit du fichier d'en-tête standard, qui définit les nouveaux ID de ressources.
    L'éditeur de ressources Visual C++ lit et met à jour ce fichier.

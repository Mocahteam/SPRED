Fonction permettant de découper une chaîne de caractères en se basant sur un délimiteur.

Cette fonction prend en entrée une chaîne de caractère s et construit un vecteur de chaînes de caractères contenant les différentes parties de s qui sont séparées par le délimiteur delim dans la chaîne. Si la chaîne de caractères ne contient aucune occurrence du délimiteur, alors le vecteur renvoyé ne contiendra qu'un seul élément : la chaîne elle-même.

Parameters

delimle délimiteur utilisé pour le découpage de s.
Returns

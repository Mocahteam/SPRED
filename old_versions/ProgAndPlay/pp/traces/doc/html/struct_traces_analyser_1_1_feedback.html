int TracesAnalyser::Feedback::level

Variable prenant la valeur de l'attribut 'level' associé à la balise <feedback> dans le fichier XML de définition de feedbacks. Cet attribut est optionnel. S'il n'est pas défini, cette variable prend la valeur -1 et ne sera pas utilisée lors de l'analyse. Dans le cas, où il est défini, sa valeur est utilisée pour associer un niveau aux traces learner_spt et expert_spt.
